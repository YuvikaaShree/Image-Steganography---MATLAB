# Image Steganography in Defence using MATLAB

## 📌 Overview

This project implements image steganography techniques to securely hide sensitive data within digital images using MATLAB. It ensures covert communication by embedding information without affecting image quality.

## ⚙️ Tools & Technologies

* MATLAB
* Image Processing
* LSB (Least Significant Bit) Technique

## 🚀 Features

* Data embedding inside images
* Secure extraction of hidden messages
* Minimal distortion in image quality
* Suitable for defence and secure communication

## 🧠 Methodology

* Convert message into binary
* Embed using LSB technique
* Extract using reverse decoding process

## 📊 Applications

* Defence communication
* Digital watermarking
* Secure data transfer

## ▶️ How to Run

1. Open MATLAB
2. Run embedding script
3. Input image and message
4. Run extraction script to retrieve message
